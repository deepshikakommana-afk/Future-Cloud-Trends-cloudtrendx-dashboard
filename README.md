# CloudTrendX — Future Cloud Intelligence Dashboard

CloudTrendX is a futuristic AI-powered cloud analytics dashboard designed to visualize and analyze future cloud computing trends, enterprise infrastructure growth, AI adoption, hybrid cloud systems, and cloud market intelligence.

The platform provides a modern enterprise-style dashboard experience with interactive charts, AI prediction modules, infrastructure activity monitoring, and cloud provider analytics.

---

# Features

* Futuristic SaaS-style dashboard UI
* AI-powered cloud trend analytics
* Interactive Bar, Line, and Doughnut charts
* Cloud provider market share analysis
* AI prediction and forecasting panels
* Infrastructure activity monitoring
* Responsive dashboard design
* Full-stack architecture using React and Express
* Modern Tailwind CSS styling
* REST API integration

---

# Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/cloudtrendx-dashboard.git
```

---

# Frontend Setup

```bash
cd cloudtrendx-client
npm install
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```

---

# Backend Setup

```bash
cd cloudtrendx-server
npm install
node server.js
```

Backend runs on:

```bash
http://localhost:5000
```
